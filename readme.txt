To start and run TicTacToe:

1. open 3035573628 file
2. In Terminal, type javac *.java  to compile the java program
3. type java TicTacToe

To start another player/ have another client:
open another terminal window, open the file and type java TicTacToe again

Remember to drag the window to the other side, because the two TicTacToe windows overlapp each other.

Whether the player wins/loses/tie is shown on the message title located below the menuBar.
